# Terminal Games

This repository contains interactive games playable directly in the terminal, written in C++ and Python. Each game demonstrates clean, modular, and procedural/object-oriented design — ideal for learning or showcasing terminal-based logic, input handling, and visual rendering.

## Project Structure

```
Hangman/
├── c++/
│   ├── hangman
│   └── tic_tac_toe
└── python/
    ├── hangman
    └── tic_tac_toe
```
