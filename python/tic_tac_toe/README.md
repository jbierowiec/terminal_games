# Python Tic Tac Toe Game

A simple terminal-based Tic Tac Toe game written in Python, designed in a procedural programming style. This program demonstrates fundamental Python concepts such as input handling, conditional logic, and array traversal.

## Project Structure

```
python_tic_tac_toe/
└── tictactoe.py
```

## Example Gameplay

```

===== TIC TAC TOE =====

Player 1: X
Player 2: O


   |   |
---+---+---
   |   |
---+---+---
   |   |


Player X, enter your move (1-9): 1


 X |   |
---+---+---
   |   |

   |   |


Player O, enter your move (1-9): 2


 X | O |

   |   |

   |   |

   |   | X

   | O |
```

## How to Run

### 1. Ensure you have Python 3 installed

Check your version:

```bash
python3 --version
```

### 2. Run the game

```bash
python tictactoe.py
```
